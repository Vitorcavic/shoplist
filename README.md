# Shoplist

Um pequeno projeto "Shopping List" (lista de compras) feito com HTML, CSS e JavaScript.

## Descrição

Este projeto demonstra uma simples lista de compras onde você pode adicionar, marcar como comprados e remover itens. É uma aplicação front-end sem backend — basta abrir o arquivo `index.html` no navegador.

## Funcionalidades

- Adicionar itens à lista
- Marcar itens como comprados
- Remover itens
- Interface simples e responsiva (CSS minimalista)

## Como executar

1. Abra o arquivo `index.html` no seu navegador.

Ou, se preferir usar um servidor local (recomendado para desenvolvimento):

```bash
# usando Python 3
python3 -m http.server 8000
# depois abra http://localhost:8000
```

## Estrutura do projeto

- `index.html` — Página principal
- `style.css` — Estilos
- `scripts.js` — Lógica JavaScript
- `assets/` — Arquivos estáticos (imagens, ícones, etc.)
